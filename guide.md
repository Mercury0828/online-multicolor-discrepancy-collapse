## §The Idea

**Current state (anchor).** Altschuler–Tikhomirov (arXiv:2509.02432, 2025) prove the **two-color** prefix discrepancy threshold `ondisc_2 = Θ(loglog n)` for uniform d-sparse i.i.d. vectors (independent of `d`), an online-vs-offline average-case separation. They do **not** pose or solve the multicolor (k-partition) version.

**The gap / our claim (the CORRECTED direction).** `ondisc_k` is **decreasing** in `k` (in this random model — see the boundary note on non-monotonicity), running from `Θ(loglog n)` at `k=2` to `1`. The collapse to discrepancy 1 happens *no later than* `K_fresh=Θ(log n/loglog n)` (proven, P3); the *exact* discrepancy-1 threshold `K_1` is only bracketed `Ω(loglog n) ≤ K_1 ≤ O(log n/loglog n)` (B3, NOT yet `Θ(M)`). The headline is a clean **phase transition / sharp threshold** characterization with matching bounds, plus the qualitative finding that **online average-case behaves oppositely to offline worst-case** in `k`.

**Why this path stands (boundary + the partial results in hand).**
- The sibling's *increasing* conjectures (`Θ(loglog n+√log k)`, `Θ(√k·loglog n)`) are **DEAD** — for large `k` an explicit online algorithm achieves discrepancy 1 (P3 below). So the only live direction is *decreasing*.
- A real **lower bound is already proven**: `ondisc_k ≥ Ω(loglog n/k)` for all `k` (P2), and the empirically-confirmed decreasing+collapse picture (P5) anchors the curve at `Θ(loglog n)` (k=2) and `1` (k≥M).
- **Boundary / non-monotonicity (state it, do not over-claim):** `disc_{k+1} ≤ disc_k` is FALSE per-sequence (a d=2 four-edge instance has `disc_3 > disc_2`). The decrease is an **asymptotic random-model phenomenon**, not a per-sequence law. The claim is about `ondisc_k(μ_{d,n})` (the optimal online value on the random model), **not** any adversarial sequence.

**The load-bearing crux (the whole SODA-deciding question reduces to ONE lemma).** GPT-5 Pro (round 2) reduced both the matching upper and the matching lower bound to a single structural lemma:
> **(★ the crux) Profile-entropy / no-compression lemma:** for the forbidden (high-color) set `F_r(I)` of a typical coordinate `I` at discrepancy level `r`, prove `Q_r(A) := Pr(A ⊆ F_r(I)) ≤ ρ_r^{|A|}` for all `A ⊆ [k]`, *stable under adaptive online evolution*.

- If `(★)` **holds** → (upper) a direct k-color algorithm with the elementary-symmetric potential `G_i(s)=∏_c(1+s·e^{λ z_{i,c}})` (`z_{i,c}=S_{i,c}−occ_i/k`) achieves `O(loglog n/log k)`, AND (lower) a Profile-Branching argument gives `Ω(loglog n/log k)` ⇒ `ondisc_k = Θ(max{1, loglog n/log(k+1)})` (the **cascade** form) — a full SODA theorem.
- If `(★)` **fails** but a **star/junta compression** online strategy gives `O(loglog n/k)` ⇒ `ondisc_k = Θ(max{1, loglog n/k})` (the **fast** form) — still a clean threshold, still SODA.
- If compression neither yields a fast upper nor kills the cascade lower ⇒ only fixed-constant-`k` survives ⇒ RANDOM/ESA.

**Three clean outcomes; two are SODA.** Confidence (GPT-Pro): cascade 0.40 / fast 0.35 / plateau 0.10 / other 0.15. The project命题 is therefore **"prove or refute `(★)`"**, not "guess the form".

**Two unconditional barriers any attempt must respect (already proven — see §Theory B1/B2).** (B1) the naive "k-ary witness ⇒ `k^r`" lower bound **collapses** (forbidden sets share: profile `(D,…,D,0)` has `|F_D|=k−1`, two coordinates cover `[k]`); (B2) any single fixed 1-D zero-sum projection + A-T spread is **capped at `Ω(loglog n/k)`** — proving `loglog n/log k` *requires* nonlinear subset-entropy, not a fancier linear cut.

## §Formal Framework / Notation

- `R^n`; arrivals `v_1,…,v_T` i.i.d. uniform over exactly-`d`-ones 0/1 vectors, `d∈[2,(loglog n)²/logloglog n]`, `T=Θ(n)` (d=2 ⇒ random graph edges on `n` vertices).
- Online k-coloring: each `v_t` → `χ(t)∈[k]`, irrevocable. Color-class load `S_c(t)=Σ_{s≤t,χ(s)=c} v_s ∈ Z_{≥0}^n`.
- Coordinate profile `x_i(t)=(S_{i,1},…,S_{i,k})`; **pairwise range** `R_i = max_c x_{i,c} − min_c x_{i,c}`.
- `ondisc_k(μ_{d,n})` = the **optimal online** value of `max_{t,i} R_i(t)` (w.h.p./in expectation).
- `M = max_t #{s<t : supp(v_s)∩supp(v_t)≠∅}` (max intersecting-edge count) `= Θ(log n/loglog n)` w.h.p.
- `L = loglog n`. Forbidden set `F_D(i)=argmax_c x_{i,c}` when `R_i=D` (else `∅`); forcing condition (P4): support `E` forces `D+1` ⟺ `∪_{i∈E}F_D(i)=[k]`.

## §Evaluation / De-risking Numerics (the MODEL is random i.i.d. — so simulation of `μ_{d,n}` is legitimate, but measure the LEMMA, not just the curve)

- **Sim A (the lemma discriminator — primary).** Under a good online k-coloring, measure the **forbidden-set compression statistic** `Γ_r(q) = −(1/q)·log Pr_{A~Unif(C([k],q))}(A ⊆ F_r(v))` on stressed coordinates. **Pre-register:** flat-in-`q` `Γ` ⇒ `Q_r(A)≈ρ^{|A|}` ⇒ `(★)` plausible ⇒ **cascade**-favorable; `Γ` dropping fast in `q` ⇒ large forbidden sets compressed to few profiles ⇒ `(★)` false ⇒ **fast/plateau**-favorable. *(Already run this session — inconclusive at feasible `n` (few stressed coords; collapse regime for `k≳M`). So treat as directional, and as a structural probe to RE-RUN with a better near-optimal player and larger `n`/k, not just greedy.)*
