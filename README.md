# online-multicolor-discrepancy-collapse

Research project for idea `online-multicolor-discrepancy-collapse` (target venue: **SODA**).

**Goal.** In the Altschuler–Tikhomirov stochastic online Beck–Fiala model (i.i.d. uniform d-sparse binary vectors, T=Θ(n)), characterize the **multicolor (k-partition)** online prefix discrepancy `ondisc_k` as a function of `k`: it is **decreasing** (more colors help, on average), from `Θ(loglog n)` at `k=2` down to `1`, with a phase-transition / collapse. The whole matching characterization reduces to a single **profile-entropy / no-compression lemma** (controls both the upper and lower bound); a partial lower bound `Ω(loglog n/k)` (all `k`) is already in hand.

**How to run.** Open this directory in Claude Code and paste the contents of `start_up_for_claude_code.md` to begin Phase 0 (scaffold + literature kill-scan + adversarial de-risk of the load-bearing lemma, stopping at a human gate). `guide.md` is the read-only project constitution (research-attack orchestration: external-solver attack loop, audit/rebattle, kill→pivot, honesty rails).
